## 1. Интуитивное объяснение для новичка

Представь, что у тебя есть список чисел и ты хочешь найти конкретное число в этом списке. Бинарный поиск - это как искать в списке, только вместо того, чтобы проверять каждое число по очереди, мы проверяем, является ли текущее число больше или меньше предыдущего.  Мы делаем это, двигаясь "вниз" или "вверх" по списку, пока не найдем нужный элемент. Это очень эффективный способ поиска, особенно когда список большой.

## 2. Формальное описание шагов в виде списка

1. **Инициализация:** Начинаем с начального значения (например, первого элемента списка).
2. **Сравнение:** Сравниваем текущее число с целевым числом.
3. **Уменьшение диапазона:** Если текущее число меньше целевого, уменьшаем диапазон поиска (на одну позицию вниз).
4. **Повторение:** Повторяем шаги 2 и 3 до тех пор, пока не найдем целевое число или достигнем конца списка.
5. **Возврат:** Возвращаем найденное число.

## 3. Код на Python

```python
def binary_search(list_to_search, target):
  """
  Выполняет бинарный поиск в списке.

  Args:
    list_to_search: Список чисел для поиска.
    target: Число, которое нужно найти.

  Returns:
    Индекс найденного числа или -1, если число не найдено.
  """
  low = 0
  high = len(list_to_search) - 1

  while low <= high:
    mid = (low + high) // 2  # Вычисляем середину диапазона
    guess = list_to_search[mid]

    if guess == target:
      return mid  # Нашли! Возвращаем индекс
    elif guess > target:
      high = mid - 1  # Ищем в левой половине
    else:
      low = mid + 1   # Ищем в правой половине

  return -1  # Число не найдено
```

**Как работает код:**

*   `binary_search(list_to_search, target)`: Функция принимает список чисел (`list_to_search`) и целевое число (`target`).
*   `low = 0`: Устанавливаем начальный индекс поиска в начало списка.
*   `high = len(list_to_search) - 1`: Устанавливаем конечный индекс поиска в конец списка.
*   `while low <= high:`:  Цикл продолжается, пока `low` меньше или равно `high`. Это означает, что есть еще элементы для рассмотрения.
*   `mid = (low + high) // 2`: Вычисляем середину диапазона поиска.  `//` - это целочисленное деление.
*   `guess = list_to_search[mid]`: Запоминаем значение текущего элемента в середине списка.
*   `if guess == target:`: Если `guess` равен целевому числу, функция возвращает индекс `mid`.
*   `elif guess > target:`: Если `guess` больше целевого числа, значит, целевое число находится в левой половине списка.  Устанавливаем `high = mid - 1`.
*   `else:`: Если `guess` меньше целевого числа, значит, целеное число находится в правой половине списка. Устанавливаем `low = mid + 1`.
*   `return -1`: Если цикл завершается без нахождения целевого числа, функция возвращает -1.

Бинарный поиск - это способ найти элемент в отсортированном списке (или массиве) по определенному критерию. Он работает, как будто мы ищем в книге, начиная с середины. Если мы дошли до середины, и наткнулись на элемент, то мы нашли его. Если мы дошли до конца, то элемент не найден, и мы начинаем искать в следующем месте.  Ключевая идея -  поиск в середине, что значительно быстрее, чем перебирать весь список.

## 2. Формальное описание шагов в виде списка

1.  **Подготовка:** Убедитесь, что список отсортирован по указанному критерию.
2.  **Определение начальной позиции:** Выберите начальную позицию, в которой мы будем искать.
3.  **Сравнение:** Сравните значение элемента, который мы ищем, с значением элемента в текущей позиции.
4.  **Уменьшение диапазона:** Если значение элемента, который мы ищем, больше, чем значение элемента в текущей позиции, значит, элемент не находится в списке.  Уменьшите начальную позицию на 1.
5.  **Повторение:** Повторяйте шаги 3 и 4, пока не найдете элемент или не достигнете максимальной глубины поиска (которая ограничена).
6.  **Возврат:** Если элемент найден, верните его. Если элемент не найден, верните -1 (или какое-то другое значение, указывающее на отсутствие элемента).

def binary_search(list_to_search, target):
  """
  Ищет элемент в отсортированном списке методом бинарного поиска.

  Args:
    list_to_search: Отсортированный список, в котором нужно искать.
    target: Элемент, который нужно найти.

  Returns:
    Индекс элемента, если он найден, иначе -1.
  """
  low = 0
  high = len(list_to_search) - 1

  while low <= high:
    mid = (low + high) // 2  # Вычисляем середину
    if list_to_search[mid] == target:

      low = mid + 1  # Ищем в правой половине
    else:
      high = mid - 1 # Ищем в левой половине

  return -1  # Элемент не найден

## 3. Код на Python

```python
def binary_search(arr, target):
  """
  Выполняет бинарный поиск в массиве.

  Args:
    arr:  Массив (список) элементов для поиска.
    target:  Элемент, который нужно найти.

  Returns:
    True, если элемент найден, False в противном случае.
  """
  low = 0
  high = len(arr) - 1

  while low <= high:
    mid = (low + high) // 2  # Вычисляем средний индекс

    if arr[mid] == target:
      return True  # Элемент найден!
    elif arr[mid] < target:
      low = mid + 1  # Ищем в правой половине
    else:
      high = mid - 1  # Ищем в левой половине

  return False  # Элемент не найден
```

**Как работает код:**

*   Функция `binary_search` принимает массив `arr` и `target` как аргументы.
*   `low` и `high` инициализируются как начальные значения диапазона поиска.
*   Цикл `while` продолжается, пока `high` меньше или равно `low`.
*   `mid` вычисляется как средний индекс между `low` и `high`.
*   Если `arr[mid]` равно `target`, функция возвращает `True` (элемент найден).
*   Если `arr[mid]` меньше `target`, функция обновляет `low` до `mid + 1` (ищем в правой половине).
*   Если `arr[mid]` больше `target`, функция обновляет `high` до `mid - 1` (ищем в левой половине).
*   Если цикл завершается без нахождения элемента, функция возвращает `False`.
