## 1. Интуитивное объяснение для новичка

Представь, что у тебя есть список чисел и ты хочешь найти конкретное число в этом списке. Бинарный поиск – это как искать в списке, только вместо того, чтобы проверять каждое число, ты проверяешь, является ли текущее число больше предыдущего. Если да, то переходи к следующему числу, иначе возвращаемся к предыдущему.  Этот процесс повторяется до тех пор, пока не найдешь нужный элемент или не достигнешь конца списка. Это очень эффективный способ поиска в больших списках, особенно когда нужно проверить много элементов.

## 2. Формальное описание шагов в виде списка

1. **Инициализация:**  Начинаем с начала списка (или с первого элемента, если список пуст).
2. **Сравнение:** Сравниваем текущий элемент со средним элементом списка.
3. **Перемещение:**
   - Если текущий элемент *меньше* среднего:  Удаляем средний элемент и возвращаемся к следующему элементу в списке.
   - Если текущий элемент *больше* среднего:  Нашли нужный элемент! Возвращаем его позицию (индекс) и завершаем поиск.
4. **Повторение:** Повторяем шаги 2 и 3, пока не найдем нужный элемент или не достигнем конца списка.
5. **Возврат:** Если элемент найден, возвращаем его позицию.  Если элемент не найден, возвращаемся к началу списка и начинаем поиск заново с начала.

## 3. Код на Python

```python
def binary_search(list_):
    """
    Выполняет бинарный поиск в списке.

    Args:
        list_: Список чисел для поиска.

    Returns:
        Индекс нужного элемента, если найден, иначе - -1.
    """
    left = 0
    right = len(list_) - 1

    while left <= right:
        mid = (left + right) // 2  # Вычисляем средний индекс

        if list_[mid] == list_[0]: # Проверяем, равен ли элемент текущему элементу списка
            return mid  # Нашли элемент! Возвращаем его позицию

        elif list_[mid] < list_[0]:
            left = mid + 1  # Ищем в правой половине
        else:
            right = mid - 1 # Ищем в левой половине

    return -1  # Элемент не найден
```

**Как работает код:**

*   `binary_search(list_)`: Функция, принимающая список чисел.
*   `left = 0`, `right = len(list_) - 1`: Устанавливаем начальные индексы левого и правого пределов списка.
*   `while left <= right:`:  Цикл продолжается до тех пор, пока левый индекс не станет больше правого индекса.
*   `mid = (left + right) // 2`: Вычисляем средний индекс. `//` - это целочисленное деление.
*   `if list_[mid] == list_[0]:`:  Проверяем, равен ли текущий элемент ( `list_[mid]` )  значению первого элемента списка (`list_[0]`). Если да, значит, нашли нужный элемент. Возвращаем `mid`.
*   `elif list_[mid] < list_[0]:`: Если текущий элемент меньше первого, то нужно искать в правой половине списка. Устанавливаем `left = mid + 1`.
*   `else:`:  Если текущий элемент больше первого, то нужно искать в левой половине списка. Устанавливаем `right = mid - 1`.
*   `return -1`: Если цикл завершается без нахождения элемента (т.е., `left > right`), значит, элемент не найден, и возвращаем `-1`.

### 2) Формальное описание шагов в виде списка
1. **Проверка пустого списка**: Если список пустой, возвращаем `-1`.
2. **Установка границ**: Определяем левую (`left`) и правую (`right`) границы списка.
3. **Цикл поиска**:
   - Вычисляем середину: `mid = (left + right) // 2`.
   - Сравниваем значение в середине с целым числом:
     - Если равны — возвращаем индекс `mid`.
     - Если меньше — уменьшаем правую границу: `right = mid - 1`.
     - Если больше — увеличиваем левую границу: `left = mid + 1`.
4. **Завершение**: Если `left > right`, возвращаем `-1`.

def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1

### 3) Исправь код
#### Исправленная версия:
```python
def find_max(lst):
    if not lst:  # Проверка на пустой список
        return None  # или выбросить исключение, если нужно
    
    max_val = lst[0]  # Начинаем с первого элемента
    for x in lst[1:]:
        if x > max_val:
            max_val = x
    return max_val
```

#### Почему это работает?
- **Инициализация:** `max_val` устанавливается как первый элемент списка (`lst[0]`), а не `0`.
- **Обработка пустого списка:** Добавлена проверка `if not lst`, чтобы избежать ошибки при работе с пустым списком.
- **Цикл:** Сравнивает каждый последующий элемент с текущим `max_val`.

#### Примеры:
```python
find_max([-5, -3, -1])  # Вернёт -1 (правильно)
find_max([0, 1, 2])     # Вернёт 2 (правильно)
find_max([])             # Вернёт None (если список пустой)
```
